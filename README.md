# Module 10 — FuncMage  
Mastering the Ancient Arts of Functional Programming

## Objective

This module introduces the concepts of **functional programming in Python**.  
The goal is to learn how to write more expressive, reusable, and elegant code by treating functions as **first-class citizens**.

The project simulates a magical world where you become a **Function Mage**, mastering powerful techniques like lambda expressions, higher-order functions, closures, and decorators.

---

## What You Learn

- Lambda expressions for concise anonymous functions
- Higher-order functions that operate on other functions
- Closures and lexical scoping to preserve state
- Functional utilities from the `functools` module
- Memoization and performance optimization
- Decorators to extend function behavior cleanly
- Static methods and decorator usage within classes

---

## Learning Outcome

By the end of this module, you should be able to:

- Write concise transformations using **lambda expressions**
- Design **higher-order functions** that accept and return functions
- Use **closures** to maintain state without global variables
- Apply **functools tools** such as `reduce`, `partial`, and `lru_cache`
- Implement **decorators** to enhance function behavior
- Structure reusable and composable functional code

This module builds strong foundations for **advanced Python patterns, functional design, and clean code architecture**.
